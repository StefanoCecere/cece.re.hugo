# cece.re
the source of website <https://cece.re>.

## terminal commands
```
hugo new --kind post post/articles/
hugo new --kind post post/diary/2021/03/title

hugo serve --gc

# upgrade Hugo on macOS:
brew upgrade hugo

# update theme modules
hugo mod get -u ./...
```
